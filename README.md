## 🧐 평달 vs 윤달, 왜 구분해야 할까요?

양력과 음력의 날짜 차이를 메우기 위해 약 3년마다 한 번씩 돌아오는 **'윤달'**은 민속학적으로 매우 특별한 의미를 가집니다.

| 구분 | 의미 | 주요 풍습 (예시) |
| :--: | :--- | :--- |
| **평달** | 일상적인 음력의 달 | 제사, 생일 챙기기 |
| **윤달** | '신의 감시가 없는 달', 무공월(無空月) | 이사, 수의 제작, 산소 이장 |

> ⚠️ 생일이나 제사는 원래 날짜인 '평달'에 챙겨야 조상님이 찾아오신다는 관습이 있어 정확한 구분이 필수입니다.

## 💻 상당모아 계산기 활용법 (생활 체험기 요약)

블로그 포스팅에서 소개된 계산기 활용 꿀팁입니다.

### 1. 직관적인 사용법
복잡한 가입 절차 없이, 연도와 날짜만 입력하면 해당 시기가 평달인지 윤달인지 즉시 판별해 줍니다.

### 2. 중요 행사 날짜 잡기
* **이사/집수리:** 윤달 기간을 확인하여 '손 없는 달'에 안심하고 진행하세요.
* **효도 선물:** 부모님 장수를 기원하며 수의를 제작할 최적의 시기를 맞추세요.

### 3. 모바일 최적화
가족 모임이나 이동 중에도 스마트폰으로 간편하게 접속하여 어르신들께 정확한 날짜를 안내해 드릴 수 있습니다.

## 🔍 자주 묻는 질문 (FAQ)

**Q. 윤달은 몇 년마다 돌아오나요?**
A. 대략 2.7년(약 33개월)에 한 번씩 돌아오며, 19년 동안 7번의 윤달이 들어갑니다.

**Q. 윤달 생일인 사람은 언제 생일을 챙기나요?**
A. 관습적으로 윤달이 아닌 원래의 달(평달) 날짜에 생일을 챙깁니다.
